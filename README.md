# AI System Analysis Tools

This repository contains tools I’m building to analyze system behavior and explore how AI-generated responses can be evaluated and improved.

The focus is on simple, practical use cases like log analysis, incident insights, and basic anomaly detection. This is part of my ongoing effort to learn and experiment with AI/LLM evaluation alongside my background in cloud systems and incident management.

## What’s included
- Log analysis tools
- AI response evaluation
- Basic anomaly detection

## Why this project
I wanted to combine my experience in production systems and debugging with an interest in how AI systems can be evaluated and improved.
